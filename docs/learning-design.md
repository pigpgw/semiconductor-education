# 학습 설계 보완

## 핵심 방향

이 프로젝트는 쉬운 설명과 깊은 설명을 분리하지 않습니다. 모든 주제는 기초에서 시작하지만, 최종적으로는 실무 판단 기준까지 올라갑니다.

## 학습 경험 원칙

- 사용자가 처음 들어오면 콘텐츠보다 먼저 자기 위치를 알 수 있어야 한다.
- 기초 학습자는 비유와 그림만 읽어도 핵심을 얻어야 한다.
- 중급 학습자는 구조, 성능, 전력, 비용의 관계를 확인해야 한다.
- 심화 학습자는 수율, 양산성, 패키징, 고객 검증 같은 현업 판단 기준까지 따라가야 한다.
- 같은 글 안에서 난이도가 올라가야 하며, 레벨별로 완전히 다른 글로 흩어지지 않아야 한다.

## 사이트 전체 레벨 설정

- 사용자는 `/level`에서 기초, 중급, 심화 중 현재 시작점을 진단합니다.
- 추천 레벨은 읽기 안내이며 콘텐츠 접근을 막지 않습니다.
- 글 목록의 필터로 기초, 중급, 심화 글을 직접 고를 수 있습니다.
- 레벨을 모르는 사용자는 `/level`에서 진단을 먼저 진행합니다.

## 로그인 없는 설계

- 이 사이트는 계정 없이 바로 읽고 복습할 수 있어야 한다.
- 레벨 설정은 서버가 아니라 현재 브라우저에만 저장한다.
- 학습 접근성을 해치는 회원가입, 소셜 로그인, 이메일 인증, 진도 저장용 계정 기능은 MVP 범위에서 제외한다.
- 개인정보를 요구하지 않는 구조가 지방·전연령·비전공 학습자의 진입 장벽을 낮춘다.
- `/study`에서 계정 없이 로컬 체크리스트와 학습 노트를 사용해 자기 속도로 이어 간다.
- 나중에 진도 저장을 추가하더라도 localStorage, 파일 내보내기, URL 공유처럼 계정 없는 방식을 먼저 검토한다.

## 깊이 매트릭스

| 층위 | 기초 | 중급 | 심화 |
| --- | --- | --- | --- |
| 큰 그림 | 기술이 왜 필요한지 이해 | 제품군과 시스템 요구 연결 | 시장, 고객 검증, 공급 안정성까지 연결 |
| 실제 구조 | 기본 단어와 비유 | 셀, TSV, 패터닝 등 구조 설명 | 구조가 수율, 전력, 비용에 주는 영향 판단 |
| 트레이드오프 | 빠름, 저장, 미세화 구분 | 용량, 대역폭, 전력, 지연 시간 비교 | 양산성, 패키징, TCO, 로드맵 리스크 검토 |
| 증거와 출처 | 공식 자료를 쉬운 말로 변환 | 삼성/SK 주장과 기술 키워드 연결 | 홍보 문구와 엔지니어링 의미 분리 |

## 자기 레벨 판단 질문

`/level`의 질문은 지식 시험이 아니라 읽기 모드를 추천하는 장치입니다.

- CPU, 메모리, 저장장치 차이를 말할 수 있는가?
- DRAM과 NAND 차이를 휘발성/비휘발성으로 설명할 수 있는가?
- refresh, bandwidth, latency 중 하나라도 자기 말로 설명할 수 있는가?
- HBM이 왜 AI와 함께 언급되는지 말할 수 있는가?
- TSV와 적층이 성능뿐 아니라 제조 난이도와도 연결된다는 점을 이해하는가?
- EUV가 단순히 “좋은 장비”가 아니라 패터닝, 생산성, defect와 연결된다는 점을 이해하는가?

질문 수가 늘어나도 사용자는 반드시 직접 레벨을 바꿀 수 있어야 합니다. 추천은 안내일 뿐, 잠금 장치가 아닙니다.

## 글 내부 레벨 표기

각 글은 다음 표기를 사용합니다.

- `먼저 이해하기`: 전연령 학습자가 읽는 결론, 비유, 핵심 그림
- `구조로 이해하기`: 셀, 공정, 패키징, 시스템 연결
- `실무처럼 생각하기`: 병목, 트레이드오프, 양산성, 고객 요구
- `공식 자료 읽기`: Samsung/SK 자료의 표현을 근거와 주장으로 분리
- `체크 질문`: 암기보다 설명 능력을 확인

## 포트폴리오 어필 구조

- 개념 설명력: 어려운 기술을 전연령 학습자가 이해하게 만드는 능력
- 기술 구조화: DRAM, HBM, EUV를 제품·공정·시스템 관점으로 연결하는 능력
- 실무 판단: 수율, 패키징, 열, 전력, 고객 검증을 기준으로 말하는 능력
- 제품 구현력: 레벨 진단, MDX 콘텐츠, 필터, 반응형 검증을 실제 웹으로 만든 증거

## 작성 품질 게이트

새 글은 다음 질문을 통과해야 합니다.

- 초보자가 첫 문단에서 포기하지 않는가
- 중급자가 구조와 트레이드오프를 확인할 수 있는가
- 심화 학습자가 실무 제약과 공식 출처를 따라갈 수 있는가
- 공식 자료의 문장을 복사하지 않고 학습자 관점으로 재구성했는가
- 복습 질문이 정의 암기보다 설명 능력을 확인하는가
